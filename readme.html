Opis

Aplikacja konsolowa do pobierania wideo z YouTube przy użyciu najnowszej wersji yt-dlp i ffmpeg. Zoptymalizowana pod kątem YouTube.

Wymagania

  • Python 3.8 lub nowszy
  • ffmpeg (plik wykonywalny w katalogu ffmpeg/)

Instalacja

1. Utwórz środowisko wirtualne

Windows 
> python -m venv venv
> Set-ExecutionPolicy Bypass -Scope Process ## w razie problemów!
> venv\Scripts\activate
Linux/macOS 
$ python3 -m venv venv
$ source venv/bin/activate

2. Zainstaluj zależności

(venv) > pip install -r requirements.txt

3. Aktualizacja yt-dlp

(venv) > pip install --upgrade yt-dlp

4. Dodaj ffmpeg

Windows: ffmpeg/ffmpeg.exe

Linux/macOS: ffmpeg/ffmpeg (nadaj uprawnienia: chmod +x ffmpeg/ffmpeg)

Uruchomienie

Windows 
> venv\Scripts\activate
> python main.py
Linux/macOS 
$ source venv/bin/activate
$ python main.py

Użycie

  1. 1Uruchom aplikację
  2. 2
    Wklej URL YouTube, np.:
    https://www.youtube.com/watch?v=dQw4w9WgXcQ
    https://youtu.be/dQw4w9WgXcQ
  3. 3Poczekaj na pobranie (wyświetlany jest postęp)
  4. 4Plik MP4 zostanie zapisany w katalogu downloads/
  5. 5Wpisz q aby wyjść

Obsługiwane formaty URL

Format URLTyp
https://www.youtube.com/watch?v=XXXXXStandardowe wideo
https://youtu.be/XXXXXSkrócony link
https://youtube.com/shorts/XXXXXYouTube Shorts
https://youtube.com/live/XXXXXTransmisje na żywo
https://youtube.com/playlist?list=XXXXXPlaylisty

Struktura katalogów

video_downloader/
├── ffmpeg/           ← Tu umieść ffmpeg
│   └── ffmpeg(.exe)
├── downloads/        ← Pobrane filmy (MP4)
├── venv/             ← Środowisko wirtualne
├── main.py           ← Główny plik aplikacji
├── requirements.txt  ← Lista zależności
└── readme.txt        ← Dokumentacja

Rozwiązywanie problemów

"Nie znaleziono ffmpeg"

Umieść plik ffmpeg w katalogu ffmpeg/

"Sign in to confirm your age"

Wideo ma ograniczenie wiekowe - wymaga cookies

"Video unavailable"

Wideo może być zablokowane w Twoim regionie

Wolne pobieranie
